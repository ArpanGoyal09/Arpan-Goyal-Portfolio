# uploads/

Place your resume files here before starting the server:

- `Arpan_Goyal_Resume.pdf`   — PDF version of your resume
- `Arpan_Goyal_Resume.docx`  — Word version of your resume

These files are served via:
- GET /api/download/pdf
- GET /api/download/docx

> These files are excluded from git via .gitignore.
